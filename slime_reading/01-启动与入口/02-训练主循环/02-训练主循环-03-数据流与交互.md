# 训练主循环 · 数据流与交互

---

## 1. Sync 全流程时序（含 colocate offload）

```mermaid
sequenceDiagram
    participant Main as train.py 主进程
    participant RM as RolloutManager
    participant ACT as Actor RayTrainGroup
    participant CR as Critic (optional)

    Main->>Main: create_placement_groups
    Main->>RM: create_rollout_manager
    Main->>ACT: create_training_models
    opt offload_rollout
        Main->>RM: onload_weights
    end
    Main->>ACT: update_weights()
    opt offload_rollout
        Main->>RM: onload_kv
    end

    loop each rollout_id
        Main->>RM: generate.remote → ray.get
        Note over RM: Sample → train_data → ObjectRef[]
        opt offload_rollout
            Main->>RM: offload
        end
        alt use_critic
            Main->>CR: async_train
            Main->>ACT: async_train(external_data=values)
        else
            Main->>ACT: async_train
        end
        opt save_interval
            Main->>ACT: save_model
        end
        opt offload_train
            Main->>ACT: clear_memory
        end
        opt offload_rollout
            Main->>RM: onload_weights
        end
        Main->>ACT: update_weights()
        opt offload_rollout
            Main->>RM: onload_kv
        end
        opt eval_interval
            Main->>RM: eval
        end
    end
    Main->>RM: dispose
```

**Comment：** colocate 下 generate 前后 rollout 权重在 CPU/GPU 间迁移，train 侧 Megatron 也可能 offload（`offload_train`）。

## 2. Async 流水线时序

```mermaid
sequenceDiagram
    participant Main as train_async.py
    participant RM as RolloutManager
    participant ACT as Actor

    Main->>RM: generate.remote(start_id) [future₀]
    loop rollout_id
        Main->>Main: ray.get future → curr_ref
        Main->>RM: generate.remote(id+1) [prefetch]
        Main->>ACT: async_train(curr_ref)
        opt (id+1) % update_interval == 0
            Main->>Main: ray.get prefetch (drain)
            Main->>ACT: update_weights()
        end
    end
```

**Code：**

```python
## 来源：train_async.py L65-L69
        if (rollout_id + 1) % args.update_weights_interval == 0:
            rollout_data_curr_ref = ray.get(x) if (x := rollout_data_next_future) is not None else None
            rollout_data_next_future = None
            actor_model.update_weights()
```

**Comment：** prefetch 使 **train(step k)** 与 **generate(step k+1)** 重叠；update 前必须 drain 避免半完成 generate。
